## Hardware details:
	Dell
	Chromebook 3100 2-in-1
	Reg model: P30T
	Reg Type No: P30T001

## Steps
	1) Start chromebook
	2) On developer mode
	3) Execute diagnostician program
		(!) if write-protection enable, unplug battery from motherboard
	4) Install FIRM FULL
	5) Boot from usb
	6) Install linux
	7) Plug battery to motherboard ?
